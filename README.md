# BASIC-WEB-SCRAPER-FOR-HEADLINES

**Description:** Fetch and display news headlines from a website using web scraping.

**Challenges:**
 - Handle different website structures
 - Avoid getting blocked (Use headers to mimic a browser request)

---

## Tech Stack
- Java  
- Jsoup  
- IntelliJ IDEA  

---

## Project Structure

BasicWebScraper
├── src
│ └── NewsHeadlineScraper.java
├── pom.xml

---

## How to Run
1. Open the project in IntelliJ IDEA  
2. Make sure Maven dependencies are loaded  
3. Run `NewsHeadlineScraper.java`  

Headlines will be printed in the console.

---

## Note
This project is for learning purposes only and works with publicly accessible web pages.
